Linux Reference

7th August 2017 / Tags: reference

Terminology

  • text: Verbatim
  • <arg>: Required Argument
  • [arg]: Optional Argument
  • <add|delete>: Required Multiple Choice
  • [add|delete]: Optional Multiple Choice

Navigation

  • cd <directory>: Change Directory
    • Moves your current directory (AKA Working Directory) to another location.
  • pwd: Print Working Directory
    • Outputs your current directory.

  • ls: List Directory

File Operations

  • cat <file> [file2]: Concatenate files
    • Outputs file. If file2 is given, it is also output after file.
    • The command is named after concatenation, which is the act of joining two strings together.
