Site Easter Eggs

19th December 2017 / Tags: site

Some sections of the site have extended functionality that isn't perfectly tested, such as bypassing limits or modifying display properties.

Script files that have these extra functions will have an Easter Egg comment in the source code of the Coffeescript files.

For example, all pages that use the standard template with the logo at the top can be made fullscreen by running mainFullscreen() in the JavaScript console.
