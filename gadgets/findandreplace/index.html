Find and Replace

Replace Escapes

When Replace Escapes is enabled, certain escape sequences will be replaced:

  • $$: Replaced with a literal $
  • $&: Replaced with the string that matched the Find query.
  • $`: Replaced with the input that occurs before the matched string.
  • $': Replaced with the input that occurs after the matched string.
  • $n: Where n is a positive integer, inserts the nth group from the Find query. Only applies if Regex Find is enabled.
